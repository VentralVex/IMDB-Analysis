# Overview
The repository contains an R script analyzing data from IMDB. The script performs the following tasks:
* Generates a summary statistics table
* Generates a frequency table and pie chart for genre and title type, respectively
* Generates a histogram of average IMDB rating, runtime in minutes, TV episode runtime, and number of votes
* Generates a condensed scatterplot of popularity and rating
* Generates a table of average rating by genre
* Generates line graphs of average IMDB rating and average movie length by year, respectively
* Computes the average IMDB rating increase of successive TV show episodes and visualizes this as a histogram
* Uses a Multiple Linear Regression (MLR) and XGBoost to predict IMDB rating
* Infers the feature combination that maximizes IMDB rating for movies and TV shows, respectively
